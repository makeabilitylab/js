# Fonts

## Mountains of Christmas

`MountainsOfChristmas-Regular.ttf` — used for the holiday greeting text in this
demo (`sketch.js`).

- **Designer:** Font Diner, Inc.
- **Source:** [Google Fonts](https://fonts.google.com/specimen/Mountains+of+Christmas)
  / [google/fonts](https://github.com/google/fonts/tree/main/apache/mountainsofchristmas)
- **License:** Apache License 2.0 — see `MountainsOfChristmas-LICENSE.txt`.
  Free to use and redistribute, including commercially, so it's safe to commit
  to this public repo.

It replaced a previous personal-use-only font (which could not be committed for
licensing reasons). The greeting falls back to the generic `cursive` family if
this font fails to load.
